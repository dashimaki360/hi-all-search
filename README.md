# hi-all-search
search all you say

# quick start
`python getImage.py`

# use as lib
```
import getImage
words = ['tomato', 'gorira', 'iphone', 'shimane']
for word in words:
    getImage.showGoogleImgSearch(word)
```

# 処理の流れ
音声

↓ (まだない)

word

↓  (getImage.py)

画像
